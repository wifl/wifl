WIFL Test Examples

These are examples of the bogus API.

A client sends a GET /bogus/dogs/3?apikey=abc123, and recieves back a 200 OK response, with body { "name": "Rover" }.

A client sends a POST /bogus/dogs?apikey=abc123 with body { "name": "Fido" }. and recieves back a 201 OK response, with Location http://api.example.com/bogus/dogs/52 and body { "name": "Fido" }.

All examples have Accept application/json , Content-Type application/json , Host api.example.com.
